# Financial Consultant Landing Page

A responsive landing page for a financial consulting service, built with HTML and CSS. 
This project focuses on layout structure, Flexbox, and clean UI design.

---

## Features

- Responsive layout using Flexbox
- Clean and modern UI design
- Service cards with hover effects
- Structured sections (Hero, Services, Benefits, Contact)
- Simple and readable typography

## Technologies Used

- HTML5
- CSS3 (Flexbox)

## Project Structure

├── index.html
└── styles.css

## Purpose

This project was created as part of my front-end learning journey to:

- Practice responsive layouts
- Improve CSS struture and organization
- Build real-word style landing pages
- Prepare for professional web development projects
